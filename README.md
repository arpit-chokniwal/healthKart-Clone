## Run Locally

Clone the project

```bash
  git clone https://github.com/fabpot11/healthKart-Clone.git
```

Go to the project directory

```bash
  cd healthKart-Clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
